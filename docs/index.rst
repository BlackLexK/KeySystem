Описание
--------

Приложение для автоматизации учёта выдачи и возврата ключей от помещений.

Реализовано на Python с использованием tkinter и SQLite.


Установка
---------

.. code-block:: bash

   pip install -r requirements.txt
   pip install -e .


Запуск
------

.. code-block:: bash

   python main.py
