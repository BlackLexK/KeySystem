# Система учёта выдачи и возврата ключей

## Описание проекта

Данный проект представляет собой desktop-приложение для автоматизации процесса учёта выдачи и возврата ключей в организации.  
Система позволяет контролировать доступ сотрудников к помещениям, фиксировать операции выдачи и возврата ключей, отслеживать нарушения сроков возврата и формировать журнал операций.

Приложение разработано на языке Python с использованием графической библиотеки Tkinter и базы данных SQLite.

---

## Основные возможности

- авторизация пользователей;
- разграничение прав доступа;
- управление сотрудниками;
- управление помещениями;
- управление ключами;
- регистрация выдачи ключей;
- регистрация возврата ключей;
- просмотр журнала операций;
- учёт нарушений и просрочек;
- фильтрация данных;
- экспорт отчётов в Excel.

---

## Используемые технологии

- Python 3
- Tkinter
- SQLite
- OpenPyXL
